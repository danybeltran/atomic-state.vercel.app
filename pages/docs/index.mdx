# Getting started

### Installation

Using yarn

```bash copy
yarn add atomic-state
```

Or with npm

```bash copy
npm install --save atomic-state
```

### Quick start

**Example**: A simple counter app

```jsx
import { atom, useAtom } from 'atomic-state'

const countState = atom({
  key: 'count',
  default: 0 // initial value
})

function Counter() {
  const [count, setCount] = useAtom(countState)

  return (
    <div>
      <h2>Count: {count}</h2>
      <button
        onClick={() => {
          setCount(count + 1)
        }}
      >
        Increase
      </button>
    </div>
  )
}

export default function App() {
  return (
    // The AtomicState root
    <AtomicState>
      <Counter />
    </AtomicState>
  )
}
```

### Suggestions

When creating an atom, it's a good idea to indicate it will contain some state by adding `State` after its friendly name. However, you are free to use any convention you want

In the example above, the `count` state is stored in the `countState` atom:

```jsx
import { atom } from 'atomic-state'

const countState = atom({
  key: 'count',
  default: 0
})
```
