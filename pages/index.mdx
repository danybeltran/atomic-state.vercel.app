  # Atomic State
  <p>Atomic State is a state management library for React.</p>
</div>

### Overview

Atomic State works very similarly to `useState`, while providing features that make state management easier.

### Features:

- Incrementaly adoptable
- TypeScript-ready
- Server-Side Rendering
- React Native
- Effects
- Actions
- Selectors (derived states)
- Persistence

and [more](/docs/api)!
